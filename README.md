# jitlib

A crappy lib for jitting a language that doesn't exist.

## Features

✅ Basic Turing completeness

❌ A human writeable scripting language

✅ 256 bytes of RAM

❌ 64 bit registers

✅ 8 bit registers

❌ Virtual registers

✅ The ability to call out from jitted code

❌ ARM backend support

✅ x64 System-V backend support

❌ Windows support
